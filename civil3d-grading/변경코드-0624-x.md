# DH.Grading 변경 코드 — 0624-x (전체 재설계 1단계: 통합지표면만)

> JACK 지시: 기존(통합 + 외곽트림) 방식 폐기. 1단계로 **'+1단까지 더한 통합지표면'만** 생성하고 멈춤.
> daylight 추출·외곽트림·정지경계선(초록)·토량지표면 **전부 제거**. JACK이 직접 [지표면 사이 최소거리]로
> 교선을 확인해 다음 방식을 정함.
> 빌드 오류 0 · 테스트 58 통과. 이전 `변경코드-0624-w.md`는 폐기된 방식이라 무시.

## 1) GradingEngine.cs — '+1단 버퍼' 다시 포함 (정지면이 원지반을 양쪽으로 넘김)

`GradingEngine.Run()` 외부(비탈) 셀 처리부. daylight 너머 1단 버퍼(`if (diff < -p.BenchHeight) continue;`로 컷)까지
표면 점에 포함 → 교선(최소거리) 추출 가능.

```csharp
// (앞부분 동일) … d, proposed, diff 계산 후:
// 재설계 1단계: JACK 지시 '+1단까지 더한 통합지표면'. daylight 너머 1단 버퍼까지 표면에 포함.
result.Points.Add(new Point3(x, y, proposed));
result.DiffPoints.Add(new Point3(x, y, proposed - gCell));
dist[ix, iy] = d;
if (diff > 0)
{
    cons[ix, iy] = true;
    zField[ix, iy] = proposed;
    result.SlopeCellCount++;
    AccumulateVolume(result, ground, x, y, proposed, cellArea);
}
```

## 2) CreateGradingCommand.cs — 통합지표면만 만들고 종료

tr1에서 `BuildGridSurface`로 표면만 생성하고, 이후 daylight/트림/정지경계선/토량 단계를 **전부 삭제**.

```csharp
// ── tr1: '+1단까지 더한 통합지표면'만 생성(재설계 1단계) ──
using (Transaction tr = db.TransactionManager.StartTransaction())
{
    boundary = BoundaryReader.Read(tr, rPoly.ObjectId);
    if (boundary.Count < 3) { /* 경고 후 return */ }

    var tin = (TinSurface)tr.GetObject(rSurf.ObjectId, OpenMode.ForRead);
    var ground = new Civil3dGroundSurface(tin);
    p = BuildParams(boundary, ground);

    result = GradingEngine.Run(boundary, ground, p);
    if (result.BenchLoops.Count == 0) { /* 경고 후 return */ }

    DumpDebug(boundary, result, p);
    (gradeSurfId, ptCount) = SurfaceBuilder.BuildGridSurface(db, tr, result, boundary, "정지면_DHGrade");
    tr.Commit();
}

// ── 재설계 1단계: 여기서 끝. daylight·트림·정지경계선·토량 모두 생성하지 않는다.
//    JACK이 직접 '정지면_DHGrade'와 원지반으로 [지표면 사이 최소거리]를 실행해 교선을 확인. ──
string msg =
    $"통합지표면 생성 완료  [버전 0624-x·통합지표면만(재설계1)]\n\n" +
    $"· 표면 '정지면_DHGrade': 점 {ptCount:N0}개\n" +
    $"  (계획면 + 절토·성토 계단 + daylight 너머 1단 버퍼 → 원지반을 양쪽으로 넘김)\n" +
    $"· daylight / 외곽트림 / 정지경계선 / 토량: 이번 단계에서는 생성 안 함\n\n" +
    $"단높이 {p.BenchHeight}m / 소단 {p.BenchWidth}m / 절토 1:{p.CutSlope} / 성토 1:{p.FillSlope}\n\n" +
    $"→ 이제 [지표면 사이 최소거리]로 '정지면_DHGrade'와 원지반의 교선을 직접 확인하세요.";
ed.WriteMessage("\n" + msg.Replace("\n\n", "\n"));
AcadApp.ShowAlertDialog(msg);
```
