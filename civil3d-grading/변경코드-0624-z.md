## 2) 핵심 — `NtsGrading.cs` (Core)

단 링 = 경계 폴리곤을 단거리마다 Buffer 오프셋(격자 안 씀 → 톱니/빗살 없음):
```csharp
var bp = new BufferParameters {
    JoinStyle = p.MiterConvex ? JoinStyle.Mitre : JoinStyle.Round,
    MitreLimit = p.MiterLimit, QuadrantSegments = 12,
};
data.BenchRings.Add(PlatformRing(boundary, plane));      // 계획면(평지)
for (int k = 1; k <= n; k++) {
    double dTop = (k - 1) * period + slopeRun;            // 비탈 위 모서리
    double dOut = k * period;                             // 소단 바깥 모서리
    data.BenchRings.Add(OffsetRing(poly, dTop, plane, zdir * k * H, bp)); // Z=계획면±kH
    data.BenchRings.Add(OffsetRing(poly, dOut, plane, zdir * k * H, bp));
}

// OffsetRing: poly.Buffer(d, bp) → 가장 큰 외곽 링, 각 점 Z=plane.At(x,y)+zOff
```

daylight = 경계 샘플마다 바깥으로 march해 계단 정지면이 원지반과 만나는 점, Buffer(0)로 꼬임 정리:
```csharp
// 절토면(up): 원지반이 계획면 위인 곳만, 성토면: 아래인 곳만 march. 아니면 경계점으로 닫음(전환부 수렴).
double grade = baseZ + zdir * BenchProfile.Height(d, H, W, slope, N);
double diff = up ? grade - gq : gq - grade;   // ≥0 = 원지반 막 넘김 → 교선
// 모은 점 → gf.CreatePolygon(coords).Buffer(0) → 외곽 링(Z=원지반)
```

## 3) 명령 — `CreateGradingCommand.cs`

```csharp
var (cut, fill) = NtsGrading.Build(boundary, ground, p);   // 원지반은 단수+교선에만

// 단 링을 브레이크라인으로 각각 TIN 생성
var cutRes = new GradingResult();
cutRes.BenchLoops.AddRange(cut.BenchRings);
cutRes.DaylightLoops.AddRange(cutDay);          // cutDay = { cut.Daylight }
(cutId, cutLines) = SurfaceBuilder.BuildFromBenchLoops(db, tr, cutRes, boundary, "절토면_DHGrade");
// 성토면 동일 → "성토면_DHGrade"

// 각 면을 자기 daylight 외곽선으로 트림(파괴식 Outer) + 정지경계선 그림
SurfaceBuilder.TrimToGroundIntersection(tr, cutId,  cutDay,  addVertices: false);
SurfaceBuilder.TrimToGroundIntersection(tr, fillId, fillDay, addVertices: false);
```
